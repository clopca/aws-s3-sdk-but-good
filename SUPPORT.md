# Support

## Where to ask for help

- Usage questions: open a GitHub issue using the feature/question templates.
- Bugs: open a bug report with a minimal reproduction.
- Security concerns: report privately via [SECURITY.md](./SECURITY.md).

## What to include

To get faster help, include:

- Package(s) and version(s)
- Framework/runtime (Next.js, Hono, Node version)
- Reproduction steps
- Expected vs actual behavior
- Logs/stack traces

## Response expectations

Maintainers review issues as time allows. Clear reproductions and scoped reports are prioritized.
