## Prerequisites

- **Node.js** >= 20.0.0
- **React** >= 18.0.0 (only for `@s3-good/react` and `@s3-good/browser`)
- An **AWS S3 bucket** — see the [S3 Setup guide](/getting-started/s3-setup/)

Choose the packages that match your project:

## Environment variables

The SDK needs AWS credentials and bucket information at runtime. Add these to your `.env` file (or your hosting provider's environment settings):

```bash title=".env"
AWS_REGION=us-east-1
AWS_BUCKET=myapp-uploads
AWS_ACCESS_KEY_ID=AKIA...
AWS_SECRET_ACCESS_KEY=...
```

<Aside type="caution">
  Never commit your `.env` file to version control. Add it to `.gitignore`.
</Aside>

These values are passed to `createRouteHandler` on the server — see the [Quick Start](/getting-started/quick-start/) for a complete example.

## Package overview

| Package | Purpose | Environment |
|---------|---------|-------------|
| `@s3-good/core` | File router, route handlers, upload client, S3 setup utilities | Server + Client |
| `@s3-good/react` | `UploadButton`, `UploadDropzone`, `useUpload` hook, and more | Client (React) |
| `@s3-good/browser` | `S3Browser` file manager, headless hooks, preview components | Client (React) |
| `@s3-good/shared` | Shared types (`S3Config`, `UploadedFile`, etc.) and error classes | Both |

## Styles (`@s3-good/react` and `@s3-good/browser`)

Both `@s3-good/react` and `@s3-good/browser` are Tailwind v4-first.

- If your app already uses Tailwind v4, include the package components as a source so utility classes are generated:

```css title="src/app/globals.css"
@import "tailwindcss";
@source "../node_modules/@s3-good/react/dist/**/*.{js,mjs}";
@source "../node_modules/@s3-good/browser/dist/**/*.{js,mjs}";
```

- If your app does **not** use Tailwind, import the prebuilt stylesheets:

```tsx
import "@s3-good/react/styles.css";
import "@s3-good/browser/styles.css";
```

### Import paths

`@s3-good/core` exposes multiple entry points:

```typescript
// Server — createUploader, createRouteHandler, FileRouter type
import { createUploader, createRouteHandler } from "@s3-good/core/server";

// Next.js server route handlers
import { createRouteHandler } from "@s3-good/core/next";

// Next.js client helper factories (async)
import {
  generateUploadButton,
  generateUploadDropzone,
  generateNextHelpers,
} from "@s3-good/core/next-client";

// Hono — framework-specific route handler
import { createRouteHandler } from "@s3-good/core/hono";

// Client — programmatic upload functions
import { genUploader } from "@s3-good/core/client";

// SDK utilities — bucket setup and validation
import { setupBucket, validateBucketCors, S3Api } from "@s3-good/core/sdk";
```
