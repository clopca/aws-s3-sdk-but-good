Use this path when you want the shortest route from install to first successful upload.

## Minimal flow

1. Install dependencies from [Installation](/getting-started/installation/).
2. Configure S3 values from [S3 Setup](/getting-started/s3-setup/).
3. Follow [Quick Start](/getting-started/quick-start/) and copy only:
   - router definition
   - API route handler
   - one UI upload component

## Success criteria

- `pnpm dev` starts.
- Upload button renders.
- Upload succeeds and logs server callback output.

If anything fails, check [Common Errors](/getting-started/common-errors/) first.
