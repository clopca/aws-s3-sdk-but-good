Complete API reference for the `@s3-good/react` package. This package provides React components, hooks, and theming utilities for building file upload UIs.

## Components

### UploadButton

A pre-built button component for file uploads.

```typescript
import { UploadButton } from "@s3-good/react";
```

**Props:** `UploadButtonProps<TRouter, TEndpoint>`

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `endpoint` | `string` | — | Route name from your file router |
| `input` | Inferred | — | Input data for the route |
| `mode` | `"auto" \| "manual"` | `"auto"` | Upload mode |
| `disabled` | `boolean` | `false` | Disable the button |
| `className` | `string` | — | CSS class for the container |
| `appearance` | `UploadButtonAppearance` | — | Style overrides |
| `content` | `{ button?, allowedContent? }` | — | Content overrides |
| `onClientUploadComplete` | `(res: UploadFileResponse[]) => void` | — | Upload complete callback |
| `onUploadError` | `(error: UploadError) => void` | — | Error callback |
| `onUploadBegin` | `(fileName: string) => void` | — | Upload begin callback |
| `onUploadProgress` | `(progress: number) => void` | — | Progress callback |
| `headers` | `HeadersInit \| (() => HeadersInit)` | — | Custom headers |

See [UploadButton docs](/components/upload-button/) for full documentation.

## Hooks

### useUpload

React hook for file uploads with state management and progress tracking.

```typescript
import { useUpload } from "@s3-good/react";

function useUpload<TRouter, TEndpoint>(
  endpoint: TEndpoint,
  opts?: UseUploadProps,
  uploaderOpts?: { url?: string },
): UseUploadReturn
```

**Options:** `UseUploadProps`

| Option | Type | Description |
|--------|------|-------------|
| `onUploadBegin` | `(fileName: string) => void` | Called when each file starts |
| `onUploadProgress` | `(progress: number) => void` | Called with progress percentage |
| `onClientUploadComplete` | `(res: UploadFileResponse[]) => void` | Called when all files finish |
| `onUploadError` | `(error: UploadError) => void` | Called on failure |
| `headers` | `HeadersInit \| (() => HeadersInit)` | Custom headers |

**Returns:** `UseUploadReturn`

| Property | Type | Description |
|----------|------|-------------|
| `startUpload` | `(files: File[], input?) => Promise<UploadFileResponse[] \| undefined>` | Start uploading |
| `isUploading` | `boolean` | Upload in progress |
| `progress` | `number` | Progress percentage (0-100) |
| `abort` | `() => void` | Cancel current upload |
| `permittedFileInfo` | `PermittedFileInfo \| undefined` | Route config from server |

See [React Hooks docs](/client/hooks/) for full documentation.

## Factories

### generateReactHelpers

Creates typed React hooks and upload functions bound to a FileRouter.

```typescript
import { generateReactHelpers } from "@s3-good/react";

function generateReactHelpers<TRouter extends FileRouter>(
  opts?: GenerateReactHelpersOptions,
): {
  useUpload: TypedUseUpload<TRouter>;
  uploadFiles: TypedUploadFiles<TRouter>;
  createUpload: TypedCreateUpload<TRouter>;
}

interface GenerateReactHelpersOptions {
  url?: string;  // default: "/api/upload"
}
```

## Theming utilities

### resolveStyle

Resolves a `StyleField` to a `CSSProperties` object.

```typescript
import { resolveStyle } from "@s3-good/react";

function resolveStyle<TContentOpts>(
  field: StyleField<TContentOpts> | undefined,
  opts: TContentOpts,
  defaultStyle: CSSProperties,
): CSSProperties | undefined
```

### resolveClassName

Resolves a `StyleField` to a CSS class name string.

```typescript
import { resolveClassName } from "@s3-good/react";

function resolveClassName<TContentOpts>(
  field: StyleField<TContentOpts> | undefined,
  opts?: TContentOpts,
): string | undefined
```

### renderContent

Resolves content that can be static or a function.

```typescript
import { renderContent } from "@s3-good/react";

function renderContent<TContentOpts>(
  content: ReactNode | ((opts: TContentOpts) => ReactNode) | undefined,
  opts: TContentOpts,
  defaultContent: ReactNode,
): ReactNode
```

## Types

### StyleField

```typescript
type StyleField<TContentOpts> =
  | string
  | CSSProperties
  | ((opts: TContentOpts) => string | CSSProperties);
```

### FileItem

```typescript
interface FileItem {
  id: string;
  name: string;
  size: number;
  type: string;
  progress: number;
  status: FileItemStatus;
  error?: string;
  file?: File;
  url?: string;
}

type FileItemStatus = "pending" | "uploading" | "complete" | "error";
```
